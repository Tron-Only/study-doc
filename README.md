# Study Doc

A Next.js web app that fetches a Markdown file from a GitHub link and displays it as rendered Markdown in the browser. 
Designed to make it easy for students to share study notes and view documents without cloning repositories.

## Key features
- Render Markdown files from GitHub links
- Clean, readable UI for study notes
- BONUS: Flash card rogue like game inspired by Balatro

## Usage
- Provide a GitHub file URL (e.g., a raw or blob URL pointing to a Markdown file) in the app's input field or query parameter.
- The app fetches the file and renders it as Markdown in the browser.
